# `/todos`

- 파일: `app/(content)/todos/page.tsx`
- 목적: 로그인 사용자 상태와 로그아웃 동작을 확인하는 간단한 테스트 화면
- 주요 동작: 현재 사용자 객체를 그대로 출력하고 `logout` 실행
- 상태 관리: `useStore`, `useTranslations`
- 구현 메모: 업무 화면이라기보다 인증/스토어 확인용 임시 페이지에 가까움

# `/todos` 패키지 구조

```bash
app/(content)/todos/
└── page.tsx # 사용자 정보 출력 및 로그아웃 테스트 페이지
```
