# 운영 체크리스트

## 제품 운영

- 핵심 사용자 시나리오가 문서화되어 있는가
- 메뉴 구조와 실제 노출 화면이 일치하는가
- 관리자 메뉴별 담당자와 승인 프로세스가 정의되어 있는가

## 콘텐츠 운영

- 게시판 카테고리 운영 기준이 있는가
- FAQ 등록/수정/폐기 기준이 있는가
- 링크/사이트맵 최신화 주기가 정해져 있는가

## AI 운영

- `chat`, `ai-chat`, `chatbot`의 역할 구분이 명확한가
- 모델 사용 비용과 호출량 모니터링 기준이 있는가
- 지식/학습 데이터 반영 절차가 정의되어 있는가
- 부정확 응답 대응 프로세스가 있는가

## 기술 운영

- 인증 정책과 세션 만료 정책이 정리되어 있는가
- Firebase/Postgres 사용 기준이 기능별로 확정되어 있는가
- 운영 환경 변수 목록과 소유자가 정리되어 있는가

## 릴리즈 운영

- 배포 전 점검 항목이 있는가
- 기능 단위 QA 범위가 정의되어 있는가
- 회귀 테스트가 필요한 핵심 흐름이 명시되어 있는가

## 다음 문서 후보

- 화면 IA 문서
- 역할/권한 매트릭스
- 기능 우선순위 백로그
- 릴리즈 노트 템플릿
- 장애 대응 런북
