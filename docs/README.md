# HAMS-BAP Docs

HAMS-BAP 프로젝트 문서를 모아두는 루트 디렉터리입니다.

## 구성

- `pm/`
  - 프로젝트 개요, 기능 맵, 운영 체크리스트 등 PM 관점 문서
- `pages/`
  - `app` 라우트 기준 화면 문서
  - 실제 URL 기준으로 문서를 분류하며, 라우트 그룹명 `(content)`, `(content-header)`, `(siderbar-header)` 는 URL에 반영하지 않음

## 문서 원칙

- 화면 문서는 `page.tsx` 기준으로 1개 문서 작성
- 문서에는 최소한 다음 내용을 포함
  - 화면 목적
  - 주요 UI/동작
  - 사용하는 상태 관리 또는 핵심 컴포넌트
  - 구현 상태 메모
- 실험용, 목업, TODO 상태 화면은 운영 화면과 구분해서 명시

## 현재 기준

- 기준 저장소: `hams-BAP`
- 기준 일자: `2026-03-10`
- 페이지 문서 범위: `app/**/page.tsx`
